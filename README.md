# 6-opt-study-webapp

## Environment
* Python >=3.12

## Quickstart Setup
1. Clone repo
```
git clone https://github.com/koki-nagatani/6-opt-study-webapp.git
```
2. Install dependencies
If you use poetry,
```
poetry install
```
or
```
pip install -r requirements.txt
```
3. Run
If you use poetry,
```
poetry shell
streamlit run streamlit_app.py
```
or
```
streamlit run streamlit_app.py
```
